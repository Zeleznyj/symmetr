Examples of using Symmetr
=========================

Run all examples using the script run_examples.sh. This runs the script for each of the examples. These script also shows the expected output for each example and can thus serve as basic test that the program functions correctly.

These examples are described in detail in the manuscript [ arXiv:2602.21034].

Included examples:
1. Conductivity and anomalous Hall effect of Fe
2. Spin-orbit torque in CuMnAs
3. Spin-polarized current in Mn3Sn
